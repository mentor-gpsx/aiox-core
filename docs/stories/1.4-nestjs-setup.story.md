### Phase 4: Database Setup (3 hours)

**Install Supabase client:**
```bash
npm install @supabase/supabase-js
```

**Create `src/common/supabase.config.ts`:**
```typescript
import { createClient } from '@supabase/supabase-js';

const supabaseUrl = process.env.SUPABASE_URL!;
const supabaseKey = process.env.SUPABASE_ANON_KEY!;

export const supabase = createClient(supabaseUrl, supabaseKey);

// Simple health check
export async function checkDbConnection() {
  try {
    const { data, error } = await supabase
      .from('users')
      .select('count', { count: 'exact', head: true });
    
    if (error) throw error;
    return { status: 'ok', connected: true };
  } catch (error) {
    return { status: 'error', connected: false, message: error.message };
  }
}
```

**Create health check endpoint:**
```typescript
// src/health/health.controller.ts
import { Controller, Get } from '@nestjs/common';
import { checkDbConnection } from '@src/common/supabase.config';

@Controller('health')
export class HealthController {
  @Get()
  async health() {
    const dbStatus = await checkDbConnection();
    return { status: 'ok', db: dbStatus };
  }
}
```

2. **Path aliases** - Always use `@src/*`, `@modules/*` — never relative imports
3. **Tests first** - Even if simple, ensure health check test passes
4. **CI must pass** - All checks (lint, typecheck, test, build) required before merge

### Recommended Approach
1. Initialize NestJS project via CLI

3. Setup ESLint/Prettier

### Questions?

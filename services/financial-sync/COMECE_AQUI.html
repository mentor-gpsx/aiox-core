⚠️ Se Parar de Funcionar

Terminal fechou acidentalmente?
Abra novo terminal e digite:
cd c:/Users/venda/aiox-core/services/financial-sync && npm run start
Dashboard não carrega?
1. Verifique se o terminal está rodando
2. Tente abrir em outro navegador
3. Limpe o cache (Ctrl+F5)
